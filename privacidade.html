Política de Privacidade

Esta política explica como os dados informados no formulário de cotação são utilizados.

1. Dados coletados

Podemos coletar nome, telefone, e-mail, cidade, tipo de plano buscado, quantidade de vidas, prazo de contratação, faixa de investimento e observações enviadas voluntariamente.

2. Finalidade

Os dados são usados para atendimento comercial, organização da solicitação de cotação e contato por equipe da Azevedo Saúde.

3. Compartilhamento

Os dados podem ser compartilhados com corretores ou equipes comerciais responsáveis por realizar o atendimento solicitado.

4. Consentimento

O formulário exige autorização expressa para contato. Sem consentimento, a solicitação não é enviada.

5. Segurança

O projeto deve ser publicado com HTTPS, autenticação administrativa e restrição de acesso aos dados.

6. Remoção dos dados

O titular pode solicitar a exclusão ou correção dos dados pelos canais oficiais de atendimento do site.

Importante: este texto é uma base inicial e deve ser revisado por advogado antes da publicação em produção.
